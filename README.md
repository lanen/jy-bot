# jy-bot

必须先安装依赖：

slack 的api 客户端：

```
pip3 install slackclient
```

yaml 配文件
```
pip3 install pyyaml 
```

我的环境:

```
Python 3.7.4

Package       Version
------------- -------
pip           18.1
setuptools    40.6.2
PyYAML        5.1.1
slackclient   2.1.0

```


配置文件的地址：

```
/usr/local/etc/jy-bot/config.yml
```


```
token: "<Your slack access token>"
```
